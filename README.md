🎬 Rotten Tomatoes Analytics: Movie Performance & Audience Insights Dashboard
A dynamic, interactive data visualization solution built in Microsoft Power BI to analyze 16K+ movies, focusing on critic vs audience ratings, genre performance, content distribution, and seasonal audience trends—enabling data-driven insights into movie success patterns and viewer preferences.

Purpose
The Rotten Tomatoes Dashboard is an interactive Microsoft Power BI report designed to analyze 16K+ movies, enabling users to compare critic (Tomatometer) vs audience ratings, genre performance, and content distribution. It helps stakeholders such as analysts, media professionals, and decision-makers uncover trends in movie quality, audience engagement, and release patterns to support data-driven insights in the entertainment domain.

Tech Stack
The dashboard was built using the following tools and technologies:

📊 Microsoft Power BI Desktop – Primary platform used to design and develop the interactive dashboard.
📂 Power Query – Used for data cleaning, transformation, and shaping of the Rotten Tomatoes dataset.
🧠 DAX (Data Analysis Expressions) – Created calculated measures (e.g., averages, counts, Top N logic) and enabled dynamic insights.
🎛️ Interactive Features – Implemented slicers, cross-filtering, and navigation buttons for enhanced user experience.
📁 File Formats – Developed using .pbix/.pbit and exported visuals as images for presentation and portfolio use.

Data Source
Source: Maven Analytics (Rotten Tomatoes Movies Dataset)
The dataset contains detailed information on 16K+ movies, including attributes such as movie title, genre, release year, runtime, studio, Tomatometer (critic) ratings, audience ratings, and Tomatometer status (Fresh, Rotten, Certified Fresh).

Features / Highlights (Short Version)
Business Problem
Analyzing large movie datasets makes it difficult to quickly understand genre performance, rating differences, and audience trends.

Goal of the Dashboard
Built an interactive Microsoft Power BI dashboard to compare critic vs audience ratings, identify top genres, and analyze movie trends over time.

Key Visuals
KPIs: 16K+ movies, avg ratings ~60, runtime ~102 mins
Top Genres: Drama & Comedy lead
Critic vs Audience Chart: Shows rating gaps
Trend Charts: Monthly audience peaks (~350K), yearly releases (~700)
Distribution: R-rated (~38%), Rotten (~43%) dominate

Business Impact & Insights
Identified top-performing genres for content strategy
Revealed audience vs critic differences
Highlighted seasonal trends for release planning
Reduced analysis time with interactive insights
